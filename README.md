# GCC Development Environment

Этот репозиторий содержит конфигурацию для создания кастомного Docker-образа среды разработки GCC.

## Быстрый старт

Чтобы загрузить последнюю версию образа, используйте команду:

```bash
docker pull ghcr.io/itmo-asd-labin-2026/gcc-environment:v15.2
```

## Выбор тега

В репозитории используется система тегов для разграничения стабильных версий и версий для разработки:

* `v15.2` — фиксированная версия компилятора.
* `pr-1` — экспериментальная версия конкретного PR. 

## Монтирование томов

Для компиляции кода внутри контейнера, используйте монтирование директорий:

```bash
docker run -it --rm \
  -v "$(pwd)":/project \
  ghcr.io/itmo-asd-labin-2026/gcc-environment:v15.2
```
